HTML Tags Cheat Sheet

Headings

Headings from "h1" to "h6" identify blocks and sections. Every page must have one "h1"

Paragraphs

Paragraphs of the text go in "p" tags.

Lists

Unordered Lists

Unordered lists are maked with bullet points and go in "ul" tags. Each list item goes in an "li" tag.

Types of Lists

  • Unordered lists ("ul")
  • Ordered lists ("ol")

Ordered Lists

Ordered lists are numbered and go in "ol" tags. Each list item goes in an "li" tag.

Anchors

Anchor tags create links to other pages. The URL goes in the 'href' attribute. Go to DuckDuckGo to search for more information.

Setting 'target="_blank"' will open the link in the new window. MDN for more details.

Anchor tags can also create links to a new location on the same page. Instead of putting a URL goes in the "href" attribute, just use the "id" of the element you want to navigate to. This allow you to Jump to the top of the page.
